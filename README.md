# Optimization Methods in Machine Learning

This repository contains a collection of Python implementations focusing on mathematical optimization techniques, ranging from classical linear algebra and convex optimization to evolutionary algorithms.

## 🚀 Overview

Optimization is the backbone of Machine Learning. This project explores 10 key optimization problems and algorithms implemented in Python, providing a hands-on approach to understanding how models converge and solve constraints.

## 📂 Contents

The repository includes implementations for the following:

1.  **Linear Programming:** Solving constrained minimization problems using `scipy.optimize`.
2.  **Eigenvalue Decomposition (EVD):** Decomposing square matrices into eigenvalues and eigenvectors.
3.  **Singular Value Decomposition (SVD):** Implementing the $A = U\Sigma V^T$ decomposition.
4.  **Second-Order Cone Programming (SOCP):** Solving convex problems with cone constraints.
5.  **KKT Conditions:** Demonstrating Karush-Kuhn-Tucker optimality conditions for non-linear programming.
6.  **Gradient Descent:** Classic first-order iterative optimization for finding local minima.
7.  **Frank-Wolfe Algorithm:** Also known as the Conditional Gradient method for constrained optimization.
8.  **Stochastic Gradient Descent (SGD):** The core optimizer for modern Neural Networks.
9.  **Genetic Algorithm (GA):** A metaheuristic inspired by natural selection to evolve a target string.
10. **Particle Swarm Optimization (PSO):** A population-based stochastic optimization technique.

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **Libraries:** * `NumPy`: Numerical computations and Linear Algebra.
    * `SciPy`: Optimization and specialized mathematical functions.
    * `CVXPY`: Modeling and solving convex optimization problems.
    * `Matplotlib`: Visualization of convergence and loss curves.
